# Student Management System

A Python-based Student Management System built using Object-Oriented Programming.

## Features

- Add Student
- Delete Student
- Search Student
- Update Student
- Count Students
- Save to JSON
- Load from JSON
- Exception Handling

## Technologies

- Python
- JSON
- OOP
- File Handling

## Project Structure

main.py
operations.py
student.py
students.json

## How to Run

python main.py
